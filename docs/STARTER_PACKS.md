# Starter Packs

Starter packs are ready-made FolderFresh profiles for common folders and
workflows. They are meant to be imported, previewed on a temporary folder, and
then adjusted to match your own folder structure.

## How To Use

1. Open FolderFresh.
2. Go to **Profiles**.
3. Choose **Starter Packs**.
4. Select a pack to import.
5. Preview the result on a temporary folder before using it on important files.

## Included Packs

| Pack | Purpose |
| --- | --- |
| Clean Downloads | Routes common downloads into documents, installers, archives, and images. |
| Receipts by Year/Month | Sends receipts and invoices into dated folders. |
| Screenshots by Date | Sorts screenshots into monthly folders. |
| Creator Export Inbox | Routes images, videos, audio, archives, and design files from an export folder. |
| Client Intake | Splits incoming client documents, archives, images, and spreadsheets. |

## What The Packs Match

Starter packs match file names, extensions, dates, folders, paths, sizes, and file
types. They do not read the contents of documents.
