What's New in 3.0

A complete rewrite from the ground up. Everything is new.

Native Performance

Rebuilt in C# / .NET 9 for native Windows performance. No more Python runtime — just fast, efficient file automation.

WinUI 3 Interface

Modern Fluent Design with smooth animations. A Discord-inspired dark purple theme that looks great on Windows 10 and 11.

Toast Notifications

Native Windows 10/11 toast notifications. Get notified when files are organized, even when minimized to the system tray.

System Tray

Minimize to tray, close to tray, start minimized. Run FolderFresh silently in the background with quick controls from the tray menu.

Core Features

Powerful file automation with complete control

Advanced Rule Engine

Create rules with regex patterns, date ranges, file content matching, size conditions, and more. Multiple actions per rule with dynamic placeholders.

Profile System

Create multiple profiles for different workflows. Each profile stores its own rules, categories, and settings. Switch instantly between setups.

Real-Time Watching

Monitor multiple folders simultaneously. Intelligent debouncing waits for files to finish writing. Each folder can use a different profile.

Smart Categories

Auto-organize files by type when rules don't match. Customizable categories with visual icons, colors, and extension mappings.

Preview Mode

See exactly what will happen before organizing. Preview pane shows the "before and after" so you're always in control.

Undo History

Made a mistake? Revert recent file operations with the undo feature. Optionally move deleted files to Recycle Bin.
