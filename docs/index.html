Features

Everything FolderFresh offers in one tool.

Automatic Sorting

Sorts files into categories such as Documents, Images, Videos, Audio, Archives, Code, and Other.

Preview Mode

Shows exactly what will be moved before anything happens.

Undo

One click restores every file to its original location.

Safe Mode

Copies files instead of moving them when extra caution is needed.

Smart Sorting

Recognises screenshots, assignments, invoices, camera photos, messaging media, and other file patterns.

Auto-tidy

Watches a folder in real time and sorts new files the moment they appear.

Duplicate Finder

Detects duplicates using size and partial hashing for faster results.

Age & Ignore Filters

Only move older files or skip specific file types entirely.
