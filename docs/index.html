Features

Everything included in FolderFresh 1.3.0

Automatic Sorting

Sorts files into tidy categories like Documents, Images, Videos, Audio, Code, and more.

Preview Mode

Shows exactly what will be moved before any action is taken.

Undo

One click restores all moved files back to where they came from.

Safe Mode

Copies files instead of moving them for full safety.

Smart Sorting (Improved)

Detects screenshots, photos, messaging media, invoices, school work, backups, edited videos, and more.

Multi-Folder Auto-Tidy

Watch multiple folders at once and automatically organise new files.

Duplicate Finder

Find duplicate files using fast hashing.

Custom Categories

Create your own categories with custom extensions.

Disable Categories

Hide categories you don’t use.

Restore Defaults

Reset all categories back to their default state with one click.
