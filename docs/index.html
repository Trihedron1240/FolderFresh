Features

Everything included in FolderFresh 2.0.0 — A complete file automation platform

Advanced Rule Engine

Create powerful reusable rules with multi-condition matching (filename, size, date, path, metadata) and flexible actions (move, copy, rename, delete, archive).

Profile System

Create multiple automation profiles and assign a specific profile to each watched folder for complete workflow control.

Real-Time Auto-Sync

Watch multiple folders simultaneously. Each folder respects its assigned profile. Waits for files to finish writing before processing.

Smart Category Sorting

Auto-detects screenshots, camera roll, messaging media, assignments, invoices, backups, and more. Fully customizable categories.

Test & Preview Mode

Preview rule effects before applying to real files. Test mode lets you see planned actions without executing them.

PySide6 UI

Modern, complete architectural overhaul from customtkinter to PySide6 for a professional, responsive interface.

Safety & Control

Safe deletion with confirmation. Undo history to revert file operations. Pause/resume automation without losing configuration.

Activity Log

Detailed automation history for debugging. Monitor all file operations and rule executions in real-time.

System Tray Mode

Minimize to system tray while maintaining background automation. Run silently in the background.
