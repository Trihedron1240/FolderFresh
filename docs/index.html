Features

Everything included in FolderFresh 1.4.0

Profiles

Create multiple profiles with custom settings for different workflows.

Per-Folder Profile Mapping

Assign a specific profile to each watched folder for maximum control.

Auto-Tidy

Automatically organises new files in real time across multiple folders.

Smart Sorting

Detects screenshots, photos, messaging media, invoices, school work, and more.

Ignore Patterns

Exclude files using substring or wildcard patterns (e.g., *.tmp, backup, temp_*).

Don't-Move List

Lock specific files or paths so they are never touched.

Preview & Undo

See every planned move before sorting. Undo restores everything safely.

Custom Categories

Add full custom categories with extensions and renaming rules.

Safe Mode

Copy instead of moving for full safety.
