USAGE EXAMPLES
================================================================================

Using Semantic Colors:
  color = Colors.get("primary")      # Get primary color
  bg = Colors.get("bg.secondary")    # Get card background

Accessing Spacing Tokens:
  spacing = DesignTokens.Spacing.MD  # 12px
  padding = DesignTokens.Spacing.LG  # 16px

Accessing Typography:
  size = Fonts.SIZE_TITLE            # 24pt
  weight = Fonts.WEIGHT_BOLD         # 700
  height = Fonts.LINEHEIGHT_NORMAL   # 1.5

Creating Modern Components:
  btn = StyledButton("Click Me")
  input = StyledLineEdit("Type here...")
  label = TitleLabel("Page Title")

Theme Switching:
  Colors.set_theme("dark")           # Dark theme
  Colors.set_theme("light")          # Light theme

Component Showcase:
  window = ComponentShowcaseWindow()
  window.show()
