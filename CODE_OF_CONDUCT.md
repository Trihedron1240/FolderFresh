# Code Of Conduct

FolderFresh should be a practical, respectful project for people improving a
file-safety tool together.

Expected behavior:

- Be specific, kind, and constructive.
- Assume good intent while still being clear about bugs and risks.
- Respect user safety concerns, especially around file loss.
- Keep discussions focused on the product, code, docs, and release process.

Unacceptable behavior includes harassment, personal attacks, doxxing, spam, or
deliberately unsafe advice.

I may edit, hide, or remove content and participation that undermines a safe
project environment.
